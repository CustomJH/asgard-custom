# Jitter Motion Pattern Atlas

2026-07-21의 [Jitter 전체 템플릿](https://jitter.video/templates/all/)을 기준으로 만든 탐색 인덱스다. 원문은 300개 이상의 무료 템플릿을 계속 갱신하므로 최신·특정 사례가 필요하면 원문 검색을 우선하라. 아래 링크는 자산 다운로드 목록이 아니라 모션 문법을 관찰하기 위한 출처다.

## 빠른 선택

| 과업 | 먼저 볼 패턴 | 관찰할 것 |
|---|---|---|
| 제품 인터랙션 | UI 상태·피드백 | 입력과 응답의 인과, 상태 연속성 |
| 랜딩·앱 소개 | 화면·디바이스 쇼케이스 | 시선 경로, 깊이, 장면 전환 |
| 헤드라인·캠페인 | 키네틱 타이포 | 읽기 순서, 마스크, 글자 리듬 |
| 로고·아이콘 | 모프·리빌 | 실루엣 보존, 경로 연속성 |
| 지표·리포트 | 차트·카운터 | 데이터 의미와 시간 순서 |
| 히어로·루프 | 배경·앰비언트 | 끊김 없는 반복, 대비 유지 |
| 소셜·영상 | 쇼릴·타이틀 | 비트, 컷, 종횡비 |

## 사용 제한

Jitter는 템플릿을 상업 프로젝트에도 사용할 수 있다고 안내하지만, 사이트에 소개된 로고·브랜드 자산과 제3자 작업의 권리는 각 소유자에게 남아 있다고 명시한다. 따라서 Freyja는 링크된 작품을 **분석 레퍼런스**로만 취급하고, 프로젝트 고유 자산·카피·레이아웃으로 다시 만들어야 한다.
