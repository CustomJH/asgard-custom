# Asgard

Coding-agent harness — sets up multiple coding-agent environments (Claude Code, OpenCode, Codex, …). Shipped as a self-contained binary: no Node, Bun, or git required to run it.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/CustomJH/asgard-custom/main/install.sh | bash
```

Installs `asgard` to `~/.local/bin`. Then:

```bash
asgard doctor    # verify
asgard --help
```
