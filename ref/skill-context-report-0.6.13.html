문장 단위 deletion test

Give a recommended answer and its main tradeoff. 한 문장만 제거한 변형을 동일 인증 사례와 모델에서 두 번씩 실행했다.

2 / 2정본: 추천안 있음
2 / 2정본: tradeoff 있음
0 / 2삭제본: 추천안 있음
0 / 2삭제본: tradeoff 있음

관측 행동이 달라졌으므로 해당 문장은 no-op이 아니다. 정본에 유지했다. 표본이 작으므로 효과 방향의 증거이지 일반 모델 전체에 대한 통계적 보장은 아니다.
