# map: memory

- `src/asgard/memory/` — 개인 Tier0 위키 기억 패키지 (구 memory.py 단일 모듈의 분해)
- `src/asgard/memory/policy.py` — 설정·주입 게이트(inject_allowed)·위협 스캔
- `src/asgard/memory/store.py` — 파일시스템 원시 계층 — 홈 보장·락·원자 쓰기·페이지 파스/렌더
- `src/asgard/memory/index.py` — 파생 색인 — sqlite FTS·벡터 upsert·reindex
- `src/asgard/memory/recall.py` — RRF 하이브리드 검색·스냅샷 주입면·증류 넛지
- `src/asgard/memory/pages.py` — 페이지 CRUD·ingest 승인 흐름·lint
- `src/asgard/memory_semantic.py` — model2vec 시맨틱 스트림 (옵트인)
- `src/asgard/memory_context.py` — 요청 기반 zero-LLM 회수 노트 조립 (개인+프로젝트 합류점)
- `src/asgard/project_memory/` — 프로젝트 기억 패키지 (구 project_memory.py 의 분해)
- `src/asgard/project_memory/records.py` — 레코드 스키마·dataclass·검증 (순수)
- `src/asgard/project_memory/scan.py` — 아티팩트 채굴·비밀 필터·변경 경로 관측
- `src/asgard/project_memory/projection.py` — 프로젝션 manifest·lock·sync
- `src/asgard/project_memory/retain.py` — stage/claim/finish retain·완료 proposal
- `src/asgard/project_memory_backends/` — 백엔드 구현체 (Hindsight 등) 어댑터
- `src/asgard/memory_bridge/` — 프로젝트 기억 서버 브릿지 패키지 (구 memory_bridge.py 의 분해)
- `src/asgard/memory_bridge/config.py` — memory-server.json 탐색/기록·pending 2단 승인 저장소
- `src/asgard/memory_bridge/trust.py` — 백엔드 바인딩 신뢰 저장소·검증 가드
- `src/asgard/memory_bridge/client.py` — server_recall/retain 백엔드 통신 (trust 는 파사드 lazy 조회)
- `src/asgard/memory_bridge/server.py` — MCP stdio 표면
- `src/asgard/commands/memory.py` — `asgard memory` CLI — 승인·lint·ingest 운영 표면
- `src/asgard/commands/memory_dashboard/` — 기억 성좌 대시보드 패키지 (data 제공층 + HTTP 서버)
- `src/asgard/commands/memory_dashboard/data.py` — 카탈로그·그래프·로그·활동·검색 데이터 제공
- `src/asgard/commands/memory_dashboard/server.py` — host 게이트·dispatch·핸들러·기동, HTML 에셋 로드
- `src/asgard/assets/memory_dashboard.html` — 대시보드 단일 페이지 (Canvas 성좌·탭·스플래시)
